# 📊 Customer Churn Prediction using ML & TensorFlow

## 📌 Description

This project focuses on predicting whether a customer will churn (leave a service) or not using Machine Learning and Deep Learning techniques. It helps businesses identify at-risk customers and take preventive actions.

---

## 📂 Dataset

Dataset used from Kaggle:
🔗 https://www.kaggle.com/datasets/pawan1222/churnmodel?select=Churn_Modelling.csv

---

## ⚙️ Technologies Used

* 🐍 Python
* 📊 Pandas, NumPy
* 🤖 Scikit-learn
* 🧠 TensorFlow / Keras
* 📈 Matplotlib, Seaborn

---

## 🔍 Steps / Approach

1. Data Collection from Kaggle
2. Data Cleaning & Preprocessing
3. Handling Missing Values
4. Feature Encoding (Label Encoding / One-Hot Encoding)
5. Feature Scaling
6. Model Training and Evaluation
7. Performance Comparison of Models

---

## 🤖 Models Used

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)
* Neural Network (TensorFlow/Keras)

---

## 📈 Results

* Achieved accuracy of **90%+**
* Deep Learning model performed better compared to traditional ML models
* Improved performance using preprocessing and tuning techniques

---

## ✨ Features

* End-to-end ML pipeline
* Multiple model comparison
* Data visualization and insights
* Deep Learning integration using TensorFlow
* Clean and modular code

---

## 🚀 How to Run

1. Clone this repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Open the notebook:

   ```
   churn_model.ipynb
   ```
4. Run all cells
